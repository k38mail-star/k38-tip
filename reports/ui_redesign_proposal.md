# 🎨 UI重构方案 — 比赛卡片重新排版

## 问题清单

1. **稳胆标签位置不对** — 挤在中间撑坏布局
2. **闪电+胜率排版混乱** — 百分比和闪电混在一起
3. **推荐理由缺失** — 没有可交互的展示方式
4. **串关结果面板用英文队名** — 和主列表不一致

## 新卡片设计

```
┌──────────────────────────────────────────────┐
│ [✓] 🇯🇴 约旦        VS   英格兰 🏴󠁧󠁥󠁮󠁧󠁿      │
│      [AI] 🏆 主胜                     🟢稳胆   │
│      🔥 今晚 20:00          ⚡⚡⚡             │
│                           胜率 96.2%          │
│      💡 实力差距明显                            │
└──────────────────────────────────────────────┘
```

## 改动要点

| 元素 | 旧位置 | 新位置 |
|------|--------|--------|
| 🟢稳胆 | mc-info内部(右上) | 卡片右上角独立位置 |
| ⚡⚡⚡ | 和百分比同行 | 单独一行在百分比上方 |
| 胜率 | 百分比加小字 | 百分比前固定显示 |
| 💡推荐理由 | 底部固定文字 | 底部可点击图标+文字 |
